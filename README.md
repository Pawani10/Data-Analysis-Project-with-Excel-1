# Vrinda Store Annual Sales Analysis 
## Project Objective
To analyze Vrinda Store's 2022 sales data and uncover actionable insights around revenue trends, customer demographics, order fulfillment, and channel performance — with the goal of helping the store identify its most valuable customer segments and optimize sales strategy for the following year.
